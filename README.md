# Twain Automate

Twain Automate is a service platform that delivers **AI-powered automation solutions** for businesses and individuals.  
Clients don’t need any technical expertise — they simply tell us the type of automation bot they want, and we design, build, and integrate it seamlessly into their systems.

Our mission is to make automation simple, accessible, and impactful. From WhatsApp chatbots and email responders to AI customer agents and social media schedulers, Twain Automate helps organizations save time, reduce costs, and deliver exceptional customer experiences.
  

---

## Tech Stack
- **Backend:** Django / Python  
- **Frontend:** HTML, CSS, Bootstrap, JavaScript  
- **Database:** MySQL  
- **AI Integrations:** Custom chatbot flows, Tavus video agents, NLP-based responders
